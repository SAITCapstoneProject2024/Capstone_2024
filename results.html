User Guide for Power BI Dashboard

Overview

Welcome to the Sectoral Analysis and Forecast of Global Greenhouse Gas (GHG) Emissions Dashboard. This guide will help you navigate and utilize the features of the dashboard to explore and analyze GHG emissions data.

Page 1: Emissions Overview

Header Section:

  • Dashboard Title: Displays the title "Sectoral Analysis and Forecast of Global Greenhouse Gas (GHG) Emissions Dashboard".
  • GHG Type Toggle: Allows users to switch between different GHG types such as CH4 (Methane), CO2 (Carbon Dioxide), N2O (Nitrous Oxide), CO2e_20YR, and CO2e_100YR.

Overall Emission Count:

Shows the total GHG emission count (in billion tonnes) for the selected GHG type.

Total Emission by Sector, Subsector:

A hierarchical view displaying GHG emissions broken down by sector and subsector. Users can see detailed emissions for specific sectors like power, manufacturing, transportation, etc.

Users can hover over a sector of subsector to see specific emission values.

Total Emission Count by Country:

A world map visualization showing GHG emission counts by country. Larger red circles represent higher emissions.

Users can hover over a country to see specific emission values.

GHG Emission per Country:

A table listing countries with their respective total emissions and trends.

  • Country: Name of the country.
  • Total Emissions: Total GHG emissions (in billions).
  • Emission Trend: Visual representation of the emission trend over time.

GHG Emission Trend and Forecast:

A line chart displaying historical GHG emissions from 2014 to 2023 and forecasted emissions from 2024 to 2028.

Users can see the emission trends and the forecasted trajectory.

How to Use Page 1

Navigating the Dashboard:

  • Use the navigation arrows at the bottom of the dashboard to switch between different views or pages.
  • Click on the GHG type buttons (CH4, CO2, N2O, CO2e_20YR, CO2e_100YR) to change the emission type displayed across the dashboard.

Interacting with Visualizations:

  • Map Visualization: Hover over the circles on the map to see detailed emission information for each country.
  • Table Visualization: Scroll through the table to compare emissions across different countries. Hover over the trend lines to see detailed emission changes.
  • Line Chart: Hover over the data points on the line chart to see specific emission values for each year.

Filtering Data:

Use the sector and subsector tree on the right side to filter emissions data by specific sectors. Clicking on a sector or subsector will update the other visualizations to reflect the selected category.

Page 2: Emissions by Source

Source Emission Count:

Displays total GHG emissions by different gas type.

Source Emission Trend:

A line chart showing the trend of emissions by source over the years.

Emission Contribution by Source:

A table illustrating the contribution of each source to the total GHG emissions.

How to Use Page 2

Analyzing Source Emissions:

  • Use the source emission count to understand the major contributors to GHG emissions.
  • Observe the emission trend line chart to see how emissions from different sources have changed over time.
  • Compare contributions from each source using the table for insights into which sources have the highest emissions.

Final Notes:

This dashboard is designed to provide an interactive experience for users looking to analyze GHG emissions data. If you have any questions or need further assistance, feel free to reach out.
